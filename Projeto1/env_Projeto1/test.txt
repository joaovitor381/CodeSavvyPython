Meu Projeto em Python
-----------------------------

Descrição:
Meu projeto é uma aplicação de lista de tarefas simples, feita em Python, para ajudar os usuários a organizarem suas atividades diárias.

Tecnologias Utilizadas:
- Python 3.11.1

Funcionalidades:
1. Adicionar tarefa: Os usuários podem adicionar tarefas à lista, inserindo uma descrição.
2. Excluir tarefa: Os usuários podem remover tarefas da lista.
3. Listar tarefas: As tarefas são exibidas na ordem em que foram adicionadas.
